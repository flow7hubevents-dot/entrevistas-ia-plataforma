# Guía de Estilo: AI Entrevistas Pro

## Visión de Marca
AI Entrevistas Pro es una plataforma de alto rendimiento diseñada para profesionales. La estética es **Industrial**, **Black-Ops** y **Premium**.

## Paleta de Colores
- **Fondo Principal**: `#050505` (Pure Dark)
- **Fondo Elevado**: `#0c0c0c`
- **Primario**: `#3b82f6` (Pro Blue)
- **Éxito**: `#10b981` (Success Green)
- **Error/Peligro**: `#ef4444` (Danger Red)
- **Bordes**: `rgba(255,255,255,0.05)`

## Tipografía
- **Headings**: `Outfit` (Black weight para títulos de impacto)
- **Body**: `Inter` (Medium weight para legibilidad)
- **Mono**: `JetBrains Mono` o `Geist Mono` (Para elementos técnicos/God Mode)

## Iconografía
- Estilo: Outlined / Lucide React.
- Grosor de trazo: 1.5px - 2px.

## Componentes Core
- **Tarjetas**: Bordes redondeados de `2.5rem` a `3rem`.
- **Botones**: Radio de `1.5rem`, tracking widder, uppercase.
- **Desenfoques**: Uso extensivo de `backdrop-filter: blur(20px)`.
